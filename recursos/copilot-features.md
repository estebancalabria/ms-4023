# Microsoft 365 Copilot Chat (Básico) – Resumen Completo

## Introducción

Microsoft Microsoft 365 Copilot Chat (Básico) es el punto de partida para desarrollar habilidades prácticas en Inteligencia Artificial dentro del ecosistema de Microsoft 365.

Está disponible con una suscripción estándar de Microsoft 365 y permite comenzar a trabajar con IA sin requerir una licencia Premium adicional.

La experiencia está diseñada como una “rampa de entrada” hacia el uso avanzado de IA empresarial, permitiendo desarrollar hábitos y flujos de trabajo que luego pueden trasladarse directamente a Microsoft 365 Copilot Premium.

---

# Objetivo del módulo

El módulo busca enseñar cómo trabajar de forma más inteligente y colaborativa utilizando IA empresarial mediante:

# Seguridad y almacenamiento de archivos

## Consideraciones importantes

* Los archivos cargados se almacenan en OneDrive.
* Se aplican límites de carga.
* Los datos empresariales NO se utilizan para entrenar el modelo.
* La búsqueda mediante `/` puede acceder a:

  * Archivos
  * Correos electrónicos (en Outlook)

## Limitaciones de la versión Básica

Sin licencia Premium NO se incluye acceso completo a:

* Chats
* Reuniones
* Sitios organizacionales
* Datos completos de Microsoft Graph

Esto funciona como indicador de cuándo una organización puede necesitar Microsoft 365 Copilot Premium.

# Microsoft 365 Copilot Pages

Microsoft 365 Copilot Pages es una funcionalidad de Copilot Chat (Básico) que permite convertir respuestas de Copilot en áreas de trabajo persistentes, editables y colaborativas.

Su objetivo es transformar una respuesta de IA en un espacio compartido donde múltiples usuarios puedan colaborar en tiempo real.

---

# Características principales de Pages

## Crear y organizar contenido

Permite transformar respuestas del chat en páginas persistentes para:

* Generar contenido
* Organizar ideas
* Refinar información
* Centralizar trabajo colaborativo

## Mejora asistida por IA

Se pueden seguir utilizando prompts sobre la misma página para:

* Expandir contenido
* Agregar ideas
* Refinar texto
* Obtener nuevas respuestas contextualizadas

## Conversión y distribución

Las páginas pueden convertirse en:

* Documentos Word
* Presentaciones PowerPoint

Facilitando el intercambio y reutilización del contenido.

---

# Cómo funciona Copilot Pages

## Flujo básico

1. Abrir Copilot Chat (Básico)
2. Conversar normalmente con Copilot
3. Seleccionar “Editar en Pages”
4. Crear automáticamente una página editable
5. Continuar refinando contenido

---

# Edición y colaboración

## Funcionalidades

* Todo el contenido es editable
* Varias respuestas pueden agregarse a la misma página
* Se puede escribir directamente dentro del workspace
* Usando `/` se accede a tipos de contenido adicionales

## Acceso a las páginas

Las páginas pueden recuperarse desde:

* El chat original
* La pestaña Pages
* Microsoft365.com
* Documentos recientes

---

# Persistencia y almacenamiento

## Consideraciones importantes

* Las páginas se guardan indefinidamente hasta ser eliminadas.
* Si el propietario abandona la organización, las páginas pueden eliminarse tras 30 días.
* El almacenamiento utiliza tecnología de Microsoft Loop.

Microsoft Loop

---

# Compartir y colaborar en tiempo real

## Colaboración mediante menciones

Es posible colaborar usando:

* `@mentions`
* Compartición mediante links
* Componentes reutilizables

Los usuarios mencionados pueden editar contenido en tiempo real.

## Integración con aplicaciones Microsoft 365

Las páginas pueden compartirse en:

Microsoft Teams
Microsoft Outlook

* Teams

# Ejercicio práctico: Planeamiento de una cumbre de clientes

El módulo incluye un laboratorio práctico orientado a utilizar Copilot Chat (Básico) en un escenario empresarial real.

## Escenario

El usuario actúa como socio de operaciones empresariales en una empresa de consultoría que debe organizar una “Cumbre de Innovación de Clientes”.

El ejercicio busca demostrar cómo utilizar Copilot para:

* Investigación
* Brainstorming
* Visualización
* Creación de documentos
* Análisis de archivos
* Colaboración

---

# Requisito importante del laboratorio

El ejercicio debe realizarse utilizando:

## Modo Web de Copilot Chat

Incluso si se dispone de licencia Premium, el laboratorio indica cambiar manualmente al modo web para asegurar que:

* Los prompts funcionen como se espera
* La información provenga de contenido público web

---

# Tarea 1: Investigación y tendencias

Copilot se utiliza para identificar tendencias relevantes de una industria para 2025.

## Objetivo

Crear una base conceptual para definir la agenda del evento.

## Ejemplo de prompt

* “¿Cuáles son las tres principales tendencias de innovación de [industria] para 2025?”

---

# Tarea 2: Brainstorming de sesiones

A partir de las tendencias encontradas, Copilot ayuda a:

* Generar títulos de sesiones
* Crear descripciones
* Refinar tono y estilo

## Objetivo

Transformar investigación en agenda concreta para el evento.

---

# Tarea 3: Visualización y branding

Copilot genera:

* Cronogramas
* Tablas
* Escalas de tiempo
* Logotipos

## Objetivo

Crear materiales visuales para la cumbre.

## Ejemplos

* Agenda de un evento de 1 día
* Logo moderno de innovación

---

# Tarea 4: Documento de planeamiento

Copilot crea un resumen ejecutivo del evento incluyendo:

* Objetivos
* Audiencia
* Temas
* Hitos
* Métricas de éxito

Luego puede exportarse automáticamente a Word.

Microsoft Word

---

# Tarea 5: Análisis de archivos y generación de contenido

El documento generado se vuelve a cargar en Copilot para:

* Resumir información
* Generar bullets
* Crear comunicaciones internas
* Redactar emails de seguimiento

## Objetivo

Demostrar reutilización inteligente de documentos.

---

# Tarea 6: Colaboración con Copilot Pages

El contenido generado se convierte en un workspace colaborativo mediante:

Microsoft 365 Copilot Pages

## Acciones realizadas

* Editar contenido
* Insertar bloques usando `/`
* Compartir páginas
* Utilizar `@mentions`
* Colaborar en tiempo real
